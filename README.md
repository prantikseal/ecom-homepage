# Ecommerce Landing Page React + Vite

Tech Stack Used
- ReactJS
- ViteJS
- TailwindCSS
- React Slick
- SCSS


To Run the codebase
1. `npm i`
2. `npm run dev`

To Build the codebase
`npm run build`
